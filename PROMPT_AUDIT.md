## 2. ЧТО ТАКОЕ ZABOTA_PLUS (ЗАБОТА+)

Telegram-бот (@den_care_bot) — персональный AI-ассистент для девушек.

### Текущий функционал:
- 🥗 Персональная диета (Gemini LLM, КБЖУ, цели)
- 🍽 Меню на неделю с рецептами
- 💧 Трекер воды
- 😊 Трекер настроения
- 🛒 Список покупок с ценами (Хабаровск)
- 📝 Заметки
- 👤 Профиль пользователя
- 💬 Эмпатические ответы (companion)
- 📋 План на день
- 🍳 Поиск рецептов по ингредиентам (Spoonacular API)

### Онбординг: 18 шагов
- Имя, возраст, пол, вес, рост
- График работы, время подъёма/отбоя
- Уровень стресса, активность
- Хобби, бытовые дела
- Аллергии, предпочтения в еде
- Цели, бюджет
- Триггеры настроения (радует/бесит)

### Технический стек:
- Python 3.13, aiogram 3.x (Telegram Bot API)
- SQLite (aiosqlite, асинхронный)
- Gemini 2.5 Flash API (14 ключей с ротацией)
- Groq API (5 ключей, fallback)
- KeyPool v2 — production-класс управления API-ключами
- Input Normalizer (NFKC) — защита от Unicode-атак
- Prompt Firewall — защита от prompt injection
- LEVIATHAN-Core — отдельный pip-пакет
- Termux на Android (3.7 GB RAM, 106 GB диск)

### Аудитория:
- Девушки 20-35 лет
- Следят за питанием и здоровьем
- Хабаровск (цены на продукты локальные)
- 1 пользователь сейчас, потенциал — тысячи

## 4. ОГРАНИЧЕНИЯ СИСТЕМЫ

- **Termux на Android** — не сервер, работает пока телефон включен
- **14 ключей Gemini** — лимит ~1500 запросов/день (бесплатные)
- **5 ключей Groq** — fallback с меньшим качеством
- **Один разработчик** — нет команды
- **Бюджет: 0₽** — всё на бесплатных API
